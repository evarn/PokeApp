## React Native PokeApp

### О проекте

Отображается список покемонов с возможностью выбора избранных.

Также реализована игра "Who's That Pokémon?", где пользователю предстоит угадать покемона по изображению.

- React Native
- React Navigation
- Redux ToolKit
- Axios
